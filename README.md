# Mission Control

Elite quantitative trading dashboard for 0DTE/short-dated options trading.

## Deploy to Vercel

1. Push this code to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and import the repo
3. Deploy!

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Real-time market regime monitoring
- Options flow tracking
- Agent fleet management
- Trading schedule
- Memory/knowledge base
- Live log stream
